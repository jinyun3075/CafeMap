# CafeMap
## 사용환경
- vscode: 1.63.2
- node: 16.13.1
- MongoDB: 5.0.5
- 카카오 맵 api
## 실행 방법
- git clone ~~ . 깃을 받아옵니다.
- 클론 폴더로 이동
- 터미널에 'npm instal bcrypt express cors dotenv mongoose multer body-parser path fs' 입력
- view -> index.html 78번 줄 appkey= 뒤에 카카오 맵 key 입력
- 터미널 'npm start' 입력해서 서버 실행 후 view 폴더에 index 페이지 실행
